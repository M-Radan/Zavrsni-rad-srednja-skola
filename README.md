# Završni rad - Bespilotne letjelice

**Autor:** Mirko Radan  
**Škola:** Elektrotehnička škola Split  
**Vrsta rada:** Završni rad (srednja škola)  
**Status:** Obrađen i obranjen  

---

## Opis rada
Ovaj završni rad obrađuje temu **bespilotnih letjelica (dronova)** i pruža teorijski pregled različitih vrsta bespilotnih letjelica, njihovih dijelova, namjena te zakonskih i sigurnosnih regulativa u Hrvatskoj i svijetu.  

Cilj rada je pružiti pregled **tehnologije, primjene i zakonodavnog okvira** bespilotnih letjelica, te istaknuti ključne aspekte koje treba uzeti u obzir pri njihovom korištenju.

---

## Sadržaj repozitorija
- Teorijski opis bespilotnih letjelica  
- Opisi glavnih dijelova i sustava (propeleri, motori, baterije, senzori itd.)  
- Pregled zakonskih regulativa i sigurnosnih standarda   
- Zaključak i preporuke
